# 🐍 Snake Game

A classic, responsive Snake Game built with **HTML5**, **CSS3 (Grid)**, and **Vanilla JavaScript**. 

---

### 🕹️ Features
* **Dynamic Grid:** Responsive board that adapts to screen dimensions.
* **Persistent High Score:** Uses `localStorage` to save your best score.
* **Live Timer:** Tracks your survival time per session.
* **Keyboard Controls:** Smooth movement using Arrow Keys.

### 🚀 Quick Start
1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Click **Start Game** and use the arrow keys to play!
